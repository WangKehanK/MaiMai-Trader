## Prerequisites
### [Taro.js](https://github.com/NervJS/taro)
```
# To Install: 

# use npm install CLI
$ npm install -g @tarojs/cli
# OR use yarn install CLI
$ yarn global add @tarojs/cli

# To run:

# yarn
$ yarn dev:weapp
$ yarn build:weapp
# npm script
$ npm run dev:weapp
$ npm run build:weapp
```
